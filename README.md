# Python Projects

This repository contains Python projects I created while improving my programming skills and understanding Object-Oriented Programming concepts.

The main purpose of these projects is to apply Python concepts in practical scenarios and build a stronger foundation for data engineering,Artificial intelligence, and machine learning projects.

## Projects

### 🏦 Bank Account System
[View Project](./Bank_Account_oop)

A small banking system simulation built using Python OOP principles.

The project includes different account types such as Savings and Current accounts, along with features like interest calculation, overdraft handling, custom exceptions, and controlled access to account balances.

**Concepts practiced:**
- Abstract classes
- Encapsulation
- Inheritance
- Polymorphism
- Exception handling


### 🎓 Learning Management System
[View Project](./Learning_Management_System)

A simple Learning Management System that manages students and lecturers with different levels of access.

This project focuses on applying object-oriented design with role-based permissions using decorators.

**Concepts practiced:**
- Classes and objects
- Inheritance
- Decorators
- Access control
- Code organization


## Python Concepts Covered

Through these projects, I practiced:

- Abstract Base Classes (`ABC`, `@abstractmethod`)
- Encapsulation using properties
- Inheritance and method overriding
- Polymorphism
- Custom exception handling
- Special methods (`__str__`, `__repr__`, `__eq__`)
- Decorators
- Writing modular Python code
