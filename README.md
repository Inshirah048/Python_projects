# Python Projects

A collection of Python projects built to practice core Object-Oriented
Programming (OOP) concepts — abstraction, encapsulation, inheritance,
and polymorphism — as part of my transition from data analyst to
software/data engineer.

## Projects

### 🏦 [Bank Account System](./Bank_Account_oop)
A banking system with savings/current accounts, interest, overdraft
limits, and custom exception handling.

### 🎓 [Learning Management System](./Learning_Management_System)
An LMS with role-based access control — students and lecturers have
different permissions, enforced using decorators.

## Skills demonstrated
- Abstract base classes (`ABC`, `@abstractmethod`)
- Encapsulation via properties and protected attributes
- Custom exception classes
- Inheritance and polymorphism
- Dunder methods (`__str__`, `__repr__`, `__eq__`)
- Decorators for access control
